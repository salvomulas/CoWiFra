Registrieren

Sie können sich nur registrieren, wenn Sie nicht angemeldet sind.

Um sich zu registrieren, gehen Sie bitte wie folgt vor.

  1. Klicken Sie in der Hauptnavigation auf Registrieren (Das Registrierungsformular wird geöffnet)
  2. Geben Sie Ihren Wunsch-Benutzernamen ein
  3. Geben Sie Ihren Vornamen ein
  4. Geben Sie Ihren Nachnamen ein
  5. Geben Sie Ihre E-Mail Adresse ein
  6. Geben Sie Ihr Passwort ein
  7. Geben Sie Ihr Passwort erneut ein, um Tippfehler auszuschliessen
  8. Klicken Sie auf Registrieren

Sie werden daraufhin ein E-Mail mit einem Bestätigungslink erhalten. Nachdem Sie diesen Link angeklickt bzw. in Ihrem Browser eingegeben haben, wird Ihr Benutzer freigeschaltet.

Login

Sie können sich erst anmelden, nachdem Ihr Benutzer aktiviert wurde.

Um sich anzumelden gehen Sie bitte wie folgt vor.

  1. Klicken Sie in der Hauptnavigation auf Login (Das Login Formular öffnet sich)
  2. Geben Sie Ihren Benutzernamen ein
  3. Geben Sie Ihr Passwort ein
  4. Klicken Sie auf Login (Sie werden direkt auf Meine Projekte weitergeleitet)

Profil bearbeiten

Um Ihr Profil zu bearbeiten gehen Sie bitte wie folgt vor.

  1. Klicken Sie in der Hauptnavigation auf Ihren Benutzernamen
  2. Klicken Sie auf Profil bearbeiten
  3. Ändern Sie die gewüschte Eigenschaft
  4. Klicken Sie auf Speichern, um die Änderungen zu übernehmen

Meine Projekte

Unter Meine Projekte sehen Sie alle Ihre Projekte auf einen Blick. Die Übersicht ist nach den Projektrollen (Besitzer, Administrator und Benutzer) und nach Projektname sortiert.

Auf der linken Seite sehen Sie Ihre Benutzerstatistik. Diese setzt sich aus folgenden Angaben zusammen.

  • Eigene Projekte (Anzahl der Projekte, die Sie selbst erstellt haben)
  • Projekte, an denen ich mitarbeite (Anzahl der Projekte, die Sie nicht selbst erstellt haben)
  • Eigene Skizzen (Anzahl der Skizzen, die Sie erstellt haben)
  • Skizzen, an denen ich mitarbeite (Anzahl der Skizzen, die Sie nicht selbst erstellt haben)

Neues Projekt erstellen

Um ein neues Projekt zu erstellen, gehen Sie wie folgt vor.

  1. Kicken Sie auf Neues Projekt erstellen (Es öffnet sich das Formular zur Erstellung eines neuen Projektes)
  2. Geben Sie den Projektnamen ein
  3. Schreiben Sie einige Stichworte oder Sätze um das Projekt prägnant zu beschreiben (Diese können später noch angepasst werden)
  4. Erfassen Sie einige Anforderungen (Diese können später noch angepasst werden)
  5. Fügen Sie Benutzer zu Ihrem Projekt hinzu, indem Sie in der Benutzerliste auf "hinzufügen" klicken
  1. Sie können die Anzeige der Benutzer einschränken, in dem Sie die „Benutzer suchen“ Funktion nutzen
  2. Wenn Sie die Checkbox markieren, bekommt der entsprechende Benutzer Administrator-Berechtigungen

Projektansicht

Dies ist die Detailansicht eines Projektes. Auf der Linken Seite sehen Sie die wichtigsten Informationen zum Projekt.

Informationen

  • Besitzer
  • Teammitglieder (Anzahl der Teammitglieder)
  • Skizzen (Anzahl der in diesem Projekt verfügbaren Skizzen)
  • Erstellt am (Datum und Uhrzeit der Erstellung des Projektes)
  • Letzte Aktivität (Datum und Uhrzeit der letzten Aktivität)

Logbuch

Das Logbuch gibt Auskunft über die letzten 10 Änderungen am Projekt. So erhalten Sie schnell einen Überblick über den Projektstand.

Übersicht

Auf der Projektübersicht werden Ihnen der Name des Projektes und (wenn vorhanden) die Beschreibung sowie bereits erfasste Anforderungen angezeigt.

Projektinfos bearbeiten

Um die Projektinfos zu bearbeiten gehen Sie wie folgt vor.

  1. Klicken Sie auf "Projektinfos bearbeiten" (Es öffnet sich das Formular zur Bearbeitung des Projektes)
  2. Passen sie die gewünschten Bereiche an
  3. Klicken Sie auf "Speichern", um die Änderungen zu übernehmen
  • Ein Benutzer kann lediglich Anforderungen anpassen
  • Ein Administrator kann zusätzlich Benutzer zum Team hinzufügen und entfernen
  • Ein Projektbesitzer kann zusätzlich den Projeknamen und die Beschreibung ändern

Skizzen

Auf diesem Tab sehen Sie eine Liste aller verfügbaren Skizzen zu einem Projekt. Wenn noch keine Skizzen erstellt wurden, erhalten Sie diese Information in Form einer blauen Informationsbox.

Beschreibung lesen

Um die Beschreibung einer Skizze zu lesen, gehen Sie wie folgt vor.

  1. Klicken Sie auf das Chevron-Symbol neben dem Skizzennamen (Die Skizzenbeschreibung wird geöffnet)
  2. Klicken Sie eruneut auf das Chevron-Symbol, um die Beschreibung wieder zu schliessen

Neue Skizze erstellen

Um eine neue Skizze zu erstellen, gehen Sie wie folgt vor.

  1. Klicken Sie oben rechts auf "Neue Skizze" (Es öffnet sich ein Fenster)
  2. Geben Sie einen Namen und eine geeignete Beschreibung ein
  3. Klicken Sie auf „Erstellen“ (Ihre Skizze wird angelegt und Sie werden zum Designer weitergeleitet)

Skizze öffnen

Um eine neue Skizze zu öffnen, gehen Sie wie folgt vor.

  1. Klicken Sie auf den Namen der Skizze (Der Designer öffnet sich mit der entsprechenden Skizze)

Skizze bearbeiten

Um eine neue Version einer Skizze aus einer bestehenden Skizze zu erstellen, gehen Sie wie folgt vor.

  1. Klicken Sie in der Liste der Skizzen auf den Knopf "Optionen" (Es öffnet sich das Optionsmenü)
  2. Klicken Sie auf "Versionen" (Es öffnet sich die Versionsübersicht)
  3. Klicken Sie in der Liste der Versionen auf den Knopf "Optionen" (Es öffnet sich das Optionsmenü)
  4. Klicken Sie auf "Bearbeiten" um die gewüschte Version in den Designer zu laden

Skizze duplizieren

Um eine Skizze zu duplizieren, gehen Sie wie folgt vor.

  1. Klicken Sie in der Liste der Skizzen auf den Knopf "Optionen" (Es öffnet sich das Optionsmenü)
  2. Klicken Sie auf duplizieren, um eine neue Skizze aus dieser Skizze zu erstellen (Es öffnet sich das Formular zur Erstellung einer neuen Skizze)
  3. Geben Sie den Namen der neuen Skizze ein (Dieser muss sich vom Namen der aktuellen Skizze unterscheiden)
  4. Geben Sie die Beschreibung der neuen Skizze ein
  5. Klicken Sie auf "duplizieren" (Sie werden auf die Skizzenübersicht umgeleitet)
Sie können auch eine ältere Version einer Skizze duplizieren. Klicken Sie dazu auf den Knopf "Optionen" und wählen Sie den Menüpunkt "Versionen". Wiederholen Sie anschliessend die oben beschriebenen Schritte für die gewüschte Version.

Skizze löschen

Um eine Skizze zu löschen, gehen Sie wie folgt vor.

  1. Klicken Sie in der Liste der Skizzen auf den Knopf "Optionen" (Es öffnet sich das Optionsmenü)
  2. Klicken Sie auf löschen, sofern Sie über die notwendigen Berechtigungen verfügen (Es öffnet sich das Formular zur Erstellung einer neuen Skizze)
Achtung: Die Skizze wird direkt gelöscht.
Sie können eine Skizze nur löschen, wenn Sie die Skizze erstellt haben (Unabhähngig davon, ob eine Drittperson eine neuere Version der Skizze erstellt hat. Einzelne Versionen einer Skizze können nicht entfernt werden.

Rohdaten

Auf der Rohdaten-Seite werden Ihnen alle Rohdaten (hochgeladene Dateien) angezeigt, welche für dieses Projekt verfügbar sind.

Rohdaten hochladen

Um Rohdaten hochzuladen gehen Sie wie folgt vor.

  1. Klicken Sie im Tab Rohdaten auf den Knopf "Rohdaten hochladen" (Es öffnet sich das Formular zum hochladen einer Datei)
  2. Wählen Sie eine Datei in dem Sie auf "Datei auswählen" klicken
  3. Schreiben Sie eine Beschreibung zu der Datei
  4. Klicken Sie auf "hochladen"
  5. Sie können nun weitere Dateien hochladen, oder zurück zur Übersicht wechseln
  • Die maximale Dateigrösse beträgt 2 MB
  • Erlaubte Dateitypen sind: DOC, DOCX, XLS, XLSX und PDF

Team

Auf der Team-Seite werden Ihnen alle Teammitglieder angezeigt, welche diesem Projekt zugewiesen sind.

Team bearbeiten

Um das Team zu bearbeiten, gehen Sie wie folgt vor.

  1. Klicken Sie im Tab "Team" auf "Berechtigungen bearbeiten" (Es öffnet sich das Formular zur Bearbeitung der Projektinformationen)
  2. Bearbeiten Sie das Team mit Hilfe der nachstehenden Funktionen Benutzer hinzufügen und entfernen sowie Administratoren hinzufügen und entfernen

Benutzer hinzufügen und entfernen

Um Benutzer zum Projekt hinzuzufügen, gehen Sie wie folgt vor.

  1. Geben Sie den Namen des Benutzers im Suchfeld ein (Oberhalb der Benutzerliste)
  2. Fügen Sie den Benutzer zum Projekt hinzu, in dem Sie auf "hinzufügen" klicken
  3. Klicken Sie auf "Speichern", um die Einstellungen zu übernehmen

Um Benutzer vom Projekt zu entfernen, gehen Sie wie folgt vor.

  1. Klicken Sie in der Liste der Teammitglieder bei dem gewüschten Benutzer auf "entfernen" (Der Benutzer wird aus der Liste der Teammitglieder gelöscht und auf der Benutzerliste wieder eingetragen)
  2. Klicken Sie auf "Speichern", um die Einstellungen zu übernehmen

Administratoren hinzufügen und entfernen

Um Administratoren zum Projekt hinzuzufügen, gehen Sie wie folgt vor.

  1. Aktivieren Sie in der Liste der Teammitglieder die Checkbox vor dem gewüschten Benutzer, um diesem Administrator-Berechtigungen zu verleihen
  2. Klicken Sie auf "Speichern", um die Einstellungen zu übernehmen

Um Administratoren vom Projekt zu entfernen, gehen Sie wie folgt vor.

  1. Deaktivieren Sie in der Liste der Teammitglieder die Checkbox vor dem gewüschten Benutzer, um diesem die Administrator-Berechtigungen zu entziehen
  2. Klicken Sie auf "Speichern", um die Einstellungen zu übernehmen

Administration

Auf dieser Seite können Sie den Projektbesitzer ändern, sowie das gesamte Projekt löschen

Projekteigentümer ändern

Um den Projektbesitzer zu ändern, gehen Sie wie folgt vor.

  1. Klicken Sie auf die Dropdownliste (Die Liste öffnet sich)
  2. Wählen Sie den neuen Projektbesitzer aus
  3. Klicken Sie auf "Speichern", um die Änderungen zu übernehmen
Der Projektbesitzer kann nur geändert werden, wenn das Team aus mindestens zwei Mitgliedern besteht.

Projekt löschen

Um ein Projekt zu öschen, gehen Sie wie folgt vor.

  1. Klicken Sie auf der Administrations-Seite auf löschen (Es öffnet sich ein Fenster, um sicher zu gehen, dass Sie das Projekt wirklich löschen wollen)
  2. Klicken Sie auf "Projekt löschen", um das Projekt zu löschen (Sie werden auf die Projektübersicht umgeleitet)

Designer

Hier erhalten Sie nähere Informationen zur Bedienung des Designers.

Element zur Skizze hinzufügen

Ziehen Sie mit Hilfe der Maus ein Element aus dem linken Bereich des Designer und lassen Sie es auf der Zeichenfläche "fallen".

Element von Skizze entfernen

Um ein Element von Ihrer Skizze zu entfernen, gehen Sie wie folgt vor.

  1. Klicken Sie auf das zu entfernende Element (Es öffnen sich die Elementeigenschaften)
  2. Klicken Sie im Bereich "Eigenschaften" (unterhalb der Skizze) auf löschen
Alternativ können Sie auch die "Entfernen"- bzw. "Delete"-Taste verwenden.

Element eine Ebene verschieben

Um ein Element eine Ebene auf Ihrer Skizze zu verschieben, gehen Sie wie folgt vor.

  1. Klicken Sie auf das zu verschiebende Element (Es öffnen sich die Elementeigenschaften)
  2. Klicken Sie im Bereich "Eigenschaften" (unterhalb der Skizze) auf "Nach vorne", wenn Sie das Element eine Ebene weiter vorne anzeigen wollen
  3. Klicken Sie im Bereich "Eigenschaften" (unterhalb der Skizze) auf "Nach hinten", wenn Sie das Element eine Ebene weiter hinten anzeigen wollen
"Nach vorne" verschiebt ein Element vor alle anderen Elemente. "Nach hinten" verschiebt ein Element hinter alle anderen Elemente.

Elementeigenschaften ändern

Um die Eigenschaften eines Elementes zu verändern, gehen Sie wie folgt vor.

  1. Klicken Sie auf das gewüschte Element (Es öffnen sich die Elementeigenschaften)
  2. Ändern Sie den Wert der Eigenschaft
Die Änderungen werden erst aktiv, wenn Sie eine der folgenden Aktionen durchführen
  • Klicken Sie in einen leeren Skizzenbereich
  • Klicken Sie auf ein anderes Element
  • Drücken Sie die "Eingabe"- bzw. "Enter"-Taste
Wenn Sie eine Änderung gemacht haben und anschliessend direkt ein neues Element hinzufügen, werden die Änderungen nicht übernommen.

Elementfarbe ändern

Um die Farbe eines Elementes zu verändern, gehen Sie wie folgt vor.

  1. Klicken Sie auf das gewüschte Element (Es öffnen sich die Elementeigenschaften)
  2. Klicken Sie im Bereich "Eigenschaften" (unterhalb der Skizze) auf die Dropdown-Liste "Farbe" (Es öffnet sich der Colorpicker)
  3. Wählen Sie die gewüschte Farbe aus
  4. Klicken Sie auf OK um die Farbwahl anzuwenden

Elementrahmen setzen / entfernen

Um einem Element einen Rahmen hinzuzufügen oder zu entfernen, gehen Sie wie folgt vor.

  1. Klicken Sie auf das gewüschte Element (Es öffnen sich die Elementeigenschaften)
  2. Aktivieren oder deaktivieren Sie im Bereich "Eigenschaften" die Kontrollbox "Rahmen"
Diese Funktion wird nur von Figuren unterstützt

Skizze exportieren

Um eine Skizze zu exportieren, gehen Sie wie folgt vor.

  1. Klicken Sie über dem Skizzenbereich auf "Exportieren" (Es wird ein Bild erstellt)
  2. Klicken Sie in Ihrem Browser auf "Datei -> Speichern"
Alternativ können Sie auch das Kontextmenü benutzen, um das Bild zu speichern.

Skizze speichern

Um eine Skizze zu speichern, gehen Sie wie folgt vor.

  1. Klicken Sie über dem Skizzenbereich auf "Speichern" (Sie werden aufgefordert einen Kommentar zu dieser Version zu erstellen)
  2. Schreiben Sie eine kurze Beschreibung der Änderungen in dieser Version
  3. Klicken Sie auf OK (Sie werden informiert, dass die Skizze gespeichert wurde)
  4. Klicken Sie auf OK (Sie werden auf die Versionenübersicht umgeleitet)

Berechtigungen

Es gibt drei verschiedene Berechtigungstypen, welche in einem Projekt verfügbar sind

  1. Benutzer
  2. Administrator
  3. Projektbesitzer

Benutzer

Ein Benutzer hat die niedrigste Berechtigungsausprägung. Er hat Berechtigungen um folgendes zu tun.

  • Projektinfos bearbeiten (Anforderungen)
  • Neue Skizze erstellen
  • Eigene Skizzen löschen
  • Rohdaten hochladen
  • Rohdaten löschen
  • Teamliste anzeigen

Administrator

Ein Administrator hat erweiterte Berechtigungen (Alle Berechtigungen des Benutzers + weitere)

  • Projektinfos bearbeiten (Anforderungen, Team)
  • Neue Skizze erstellen
  • Eigene Skizzen löschen
  • Rohdaten hochladen
  • Rohdaten löschen
  • Teamliste anzeigen
  • Teamliste bearbeiten (hinzufügen und entfernen von Benutzern, sowie Vergabe der Administrator-Berechtigungen)

Projektbesitzer

Ein Projektbesitzer hat alle Berechtigungen (Alle Berechtigungen des Administrators + weitere)

  • Projektinfos bearbeiten (Anforderungen, Team, Projektname und Projektbeschreibung)
  • Neue Skizze erstellen
  • Alle Skizzen löschen
  • Rohdaten hochladen
  • Rohdaten löschen
  • Teamliste anzeigen
  • Teamliste bearbeiten (hinzufügen und entfernen von Benutzern, sowie Vergabe der Administrator-Berechtigungen)
  • Projektbesitzer ändern
  • Projekt löschen
